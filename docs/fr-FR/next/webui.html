Interface d'Utilisateur Web2

Verdaccio dispoe d'une interface d'utilisateur web pour afficher uniquement les paquets privés et qui peut être personnalisée.

web:
  enable: true
  title: Verdaccio
  logo: logo.png
  scope:

Toutes les restrictions d'accès définies pour la protection des pacquets s'appliquent également à l'Interface Web.
